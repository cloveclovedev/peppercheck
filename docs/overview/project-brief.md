# PepperCheck: プロジェクト概要

## 概要

PepperCheckはタスク達成モチベーションをサポートすることを目的としたプラットフォームです。
他者からタスク結果をチェックされることにより生じる責任感を利用することを特徴としています。
**Tasker**は達成したいタスクを設定し、**Referee**がプラットフォーム内のユーザーからあるアルゴリズムにしたがってマッチングされ、タスクの結果をレビューします。

## 哲学

- **責任感を燃料に**: 対人関係から生ずる責任感は、リマインダーやゲーミフィケーションよりも強力です。
- **小さく、検証可能な行動**: タスクは他者によって評価可能なものとして設計されるべきです。

## 機能 (MVP)

### タスク作成

ユーザーは以下を定義します：
- タイトル
- 説明
- **判定基準** (成功/失敗の明確な基準)
- 締切
- コミットメントフィー (Stripe経由で事前支払い)
- タスクはレフェリーにマッチングされます。

### 提出

ユーザーは締切前に結果を提出します：
- テキスト説明
- オプション画像や動画 (例: スクリーンショット)

### 判定

レフェリーは提出物をレビューし、以下を選択します：
- 成功または失敗
- 短いコメントを追加
- (後で: 相互評価システム)

### ナッジング

レフェリーはタスクごとに1回の「ナッジ」を送信してユーザーにリマインダーを送ることができます。
プッシュ/メール通知を送信します。

### 支払い

Stripeがユーザーの支払いとレフェリーへの支払いを処理します。
プラットフォームは約10%の手数料を取ります。
