== 背景

多くの人が、「自らの意思」だけでモチベーションを維持し、日々のタスクを実行することに難しさを感じています。

モチベーション維持のために「他者にタスクの達成状況をチェックしてもらう」ことは有効ですが、
家族や友人に日々のタスクをチェックしてもらうことは相手への負担が懸念され心理的ハードルが高いです。

そのため多くの人は「ひとりでは続かない」が「チェックも頼みづらい」というジレンマに陥り、継続的なタスク実行環境を持てずにいます。

== PepperCheckの目的

PepperCheckは、報酬システムを介したレビュアーマッチング機能を提供することで、このジレンマを解決します。

タスク実行者は代金を支払い、タスクの達成条件と締め切りを設定しレビュアーマッチング依頼を送ります。
システムはタスクにレビュアーをアサインします。
締め切りまでにタスク実行者はエビデンスを提出し、レビュアーはタスクの達成状況をレビューします。
レビュー完了時にシステムはタスク実行者から受け取った代金の一部をレビュアーに報酬として支払います。

以上の仕組みにより、
タスク実行者は「他者の目による社会的責任感」から強力な動機づけを得ることができます。
また適切な報酬を支払うことで持続可能なレビュアーマッチング環境を構築できます。

PepperCheckは単なるタスク管理ツールではなく、タスクを確実に「完了させる」ための実行支援プラットフォームです。
